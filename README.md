# CinoGear Spider 18DOF

A practical 18DOF spider-style hexapod robot platform under the CinoGear brand.

## Overview
This repository contains the public firmware workspace, documentation structure, BOM references, and marketplace-ready technical assets for the first CinoGear robot product.

## Repository Structure
- `firmware/` source code and controller-related files
- `docs/` public documentation
- `bom/` bill of materials
- `media/` preview images and product visuals
- `marketplace/` public-facing product support text
