# Assembly-to-BOM Notes

Key points extracted from the uploaded assembly guide:
- Assemble idler-bearing subassemblies first.
- Install femur links only after centering servos to 1500 us.
- For the leg build, servo spline indexing will not land perfectly on exact 90-degree geometry,
  so software calibration is still expected.
- The guide describes two build variants:
  1. all legs identical (author's preferred visual style),
  2. mirrored build using mirror STL variants on one side.

Potential guide inconsistencies flagged in the BOM:
- Femur idler bearing ball count is implied by the text but not explicitly listed in the parts bullet list.
- Coxa servo mount hardware line mentions M3 bolts but M2 nuts; likely a typo and should be verified.
