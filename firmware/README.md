# Firmware

This folder contains the public firmware workspace for the CinoGear Spider 18DOF robot.

## Structure
- `BotBoarduino_CH3R_PS2/` main controller sketch and related source files
- `PS2X_Library/PS2X_lib/` third-party PS2 controller library
- `config/` configuration notes and mappings
- `examples/` test or example files
